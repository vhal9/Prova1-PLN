# Prova1-PLN
- Este repositório contêm a solução para a prova 1 da disciplina de PLN. 
- O problema esta descrito no arquivo pdf.
- Arquivo Notebook/Corpora Compilation.ipynb busca os textos a serem estudados.
- Arquivo Notebook/Estudo dos dados.ipynb realiza o estudados dos dados.
- Arquivo Notebook/nlputils/lexical/preprocessing.py é um classe que funciona como biblioteca que contêm rotinas de normatização.

## Para executar:
- git clone (link deste repositório).
- entre no diretório do repositório.
- instale as bibliotecas do python com o comando: pip3 install -r requirements.txt
- execute o comando: jupyter notebook.
